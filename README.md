# E-Commerce Engine 🛒

A full-stack e-commerce platform built with the MERN stack, featuring secure authentication, product management, order processing, payment integration, and dedicated Admin and User dashboards.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/mohdmufeet/ecommerce-engine.git
cd ecommerce-engine
```

### Frontend Setup

```bash
npm install
npm start
```

### Backend Setup

```bash
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file and add:

```env
MONGODB_URI=
JWT_SECRET=
RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=
CLIENT_URL=
```
